HTML History

Check out these interesting historical facts about HTML:

HTML 1

The first draft of HTML 1 was created in 1991 by Tim Berners-Lee and it was officially released in 1993. This version provided basic structural elements and allowed the inclusion of hyperlinks.

HTML 2

HTML 2 was published in November 1995. Some notable additions to this version included tables, better support for images, alignment, margins and anchors.

HTML 3

HTML 3 was published in January 1997 as a W3C recommendation. The notable features in this version included client-side image maps, forms, input validation, and a more refined version of tables.

HTML 4

HTML 4 was published in December 1997 as a W3C recommendation. This version was a significant milestone in the development of HTML due to introducing several key features and improvements. Some notable features included strict and transitional DOCTYPES, improved forms, and scripting support.

HTML 5

HTML 5 was published in October 2014 as a W3C recommendation and is the modern standard for web development. HTML5 revolutionized web development by introducing new elements, enhanced multimedia capabilities, support for mobile devices, and the foundation for responsive design.

HTML 6

There isn't an offical specification of HTML 6, but the potential for advancements in the future of HTML are very promising. Some of the potential advancements include enhanced layout and styling, improved performance and efficiency, expanded web application capabilities, and enhanced security and privacy.
